# Airline Club Tool

A chrome extension designed for [Airline Club](https://www.airline-club.com) game.

## Features
* Adds the functionality to dispaly relationship map for each country.

## Installation

Due to limitation of google chrome not allowing extension outside their chrome web store, this tool needs to be installed manually by following the below process.

### For Windows

1. Download the [Zip archive](https://github.com/debgithub/airline-club-tool/archive/refs/heads/main.zip) of this code repostiory and extract the same.
2. Open the extension page in google chrome following the path **Menu -> More Tools -> Extensions**.
3. Enable the **Developer mode** on the top right corner.
4. Click on the **Load unpacked** option from the toolbar at the top.
5. Select the **airline-club-chrome-extension** folder from the extract.
6. Now, the extension should be loaded into chrome. Make sure to enable the same if not done automatically.
